After putting it off for a while, davidokunfolami.com is live. It's a small,
static site — plain HTML, CSS, and JavaScript, no framework, no build step —
because I wanted the site itself to be as easy to understand and maintain as
the projects it's meant to showcase.

Right now it has:

- **Projects** — [JobLens](https://github.com/Dah-vid/JobLens), a full-stack
  job application platform I built end-to-end; [Streakser](https://github.com/Dah-vid/streaks-star),
  a habit-tracking iPhone app; a speech-to-text benchmarking tool; and
  NeighbourHeat, a hackathon project analysing thermal drone imagery.
- **A CV** you can download directly.
- **This blog** — for notes that don't fit anywhere else.

The projects list and this blog are both driven by small data files
(`projects.json` and `blog/posts.json`), so adding new work later is mostly
just editing a file and pushing.

More soon, on whatever I'm building next.
